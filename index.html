Grid Runner — Adaptive Q-Learning

5×5 grid world: ε-greedy exploration, Q-table updates (α = 0.1, γ = 0.9), 500 episodes.

Environment

Train or run a greedy demo to see the agent.

Controls

Adaptation — total reward per episode

Why α and ε matter

α (learning rate) — Each step nudges Q(s,a) toward the target r + γ·max Q(s′,·). With α = 0.1, updates are conservative so estimates stay stable on this small grid.

ε (exploration) — With probability ε the agent picks a random action; otherwise it exploits the best-known action. ε starts at 1.0 and is multiplied by 0.99 after each episode, so behavior shifts from broad exploration to using what was learned.
